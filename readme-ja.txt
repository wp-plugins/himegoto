== Description ==
カスタムポストタイプを使い、秘密の投稿やキャンペーン等を投稿します。
投稿のタイトルで検索すると、ポップアップして内容が表示されます。

== Installation ==
1. `himegoto` フォルダーを `/wp-content/plugins/` アップロード。
2. プラグインメニューより有効化。
